## dotfiles for an efficient and awesome terminal
Manage dotfiles with GNU stow

OSX: `brew install stow`
Ubuntu: todo

Clone this repo: 
`
git clone https://github.com/jcaffey/dotfiles.git ~/dotfiles && cd ~/dotfiles && echo "stow [ options ] package"
`
